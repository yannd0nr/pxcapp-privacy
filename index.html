Politique de confidentialité — PXCApp

Mise à jour : 29 août 2025

Résumé

PXCApp affiche un fichier JSON de contacts (Nom, Téléphone, Date de naissance), permet de sauvegarder des contacts sélectionnés dans l’app Contacts du téléphone et d’afficher des notifications d’anniversaire. Les données restent sur votre appareil. Le développeur ne collecte ni ne partage vos données.

Données traitées

  • Contacts du fichier JSON fourni par l’utilisateur (Nom, Téléphone, Date de naissance).
  • Contacts présents sur l’appareil : lecture utilisée uniquement pour éviter les doublons avant création d’un nouveau contact.

Finalité

  • Affichage de la liste des contacts et des prochains anniversaires.
  • Création de nouveaux contacts dans l’app Contacts (au choix de l’utilisateur).
  • Notifications le jour de l’anniversaire (si autorisées).

Stockage & transmission

  • Les données sont traitées localement sur l’appareil et non transmises au développeur.
  • L’app télécharge le fichier JSON via HTTPS depuis l’URL que vous saisissez.

Partage

Aucun partage avec des tiers.

Permissions Android

  • READ_CONTACTS : vérifier les doublons dans vos contacts avant création.
  • WRITE_CONTACTS : créer les contacts que vous avez sélectionnés.
  • POST_NOTIFICATIONS (Android 13+) : afficher les rappels d’anniversaire.

Conservation & contrôle

  • Les contacts créés restent dans votre app Contacts jusqu’à leur suppression par vous.
  • Vous pouvez retirer les autorisations Android à tout moment dans les Réglages du système.

Sécurité

La connexion au fichier JSON s’effectue via HTTPS. Aucune base de données distante n’est utilisée.
